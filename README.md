# 社食おすすめ（腸タイプ×SCFA）— PoC Demo

このアプリは、個人の**腸内タイプ**と**その日の体調・気分**に合わせて、社食のメニューから「短鎖脂肪酸（SCFA）の産生を最大化する」最適な組み合わせを提案するデモアプリです。

## 🌟 特徴
- **科学的アプローチ**: 単なる栄養計算ではなく、腸内細菌による発酵（SCFAスコア）を軸にしたレコメンド。
- **直感的なUI**: チャット形式で、迷わず今日のセットを決定。
- **データ蓄積**: PoC用として、選択ログや食後のリアクションを保存可能。

## 🧬 ロジックの概要
ユーザーの腸内タイプ（Type A〜F）に基づき、各食材が持つ「発酵基質としてのポテンシャル」をスコアリングしています。
- **主菜の選択**: SCFA寄与度が高いメインを優先。
- **小鉢の補完**: 主菜で不足する発酵基質を小鉢で補う「コンボ型」の計算ロジックを搭載。

## 🚀 使い方
1. 自分の「腸内タイプ」を選択。
2. 「午後の集中」「お腹にやさしく」など、今の気分を選択。
3. 提案されたセットを確認し、「このまま行く」または「代替にする」を決定。
